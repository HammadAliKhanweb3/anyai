# AnyAI 🤖

AnyAI is a full-stack AI chatbot platform designed to interact with **multiple AI models** for a wide range of tasks such as conversation, assistance, and experimentation with different AI capabilities.

The project is structured with a **frontend** and **backend**, and scalable AI integrations.

---

## ✨ Features

- 🔗 Connect and interact with multiple AI models  
- 🧠 Unified AI chatbot interface  
- 🔐 Authentication system  
- 🌐 Full-stack architecture (Frontend + Backend)  
- ⚡ Modern JavaScript tooling (Bun / Node.js)  
- 🧩 Modular and extensible codebase

## 🚀 Getting Started

### Prerequisites

Make sure you have the following installed:

- **Node.js** (v18+ recommended) or **Bun**
- **Git**

---

### Installation

Clone the repository:

```bash
git clone https://github.com/HammadAliKhanweb3/anyai.git
cd anyai


Install dependencies:

bun install
# or
npm install

▶️ Running the Project
Backend
cd backend
bun run dev
# or
npm run dev

Frontend
cd frontend
bun run dev
# or
npm run dev


Make sure environment variables are properly configured before running the app.

🔐 Environment Variables

Create a .env file in the backend directory and configure variables such as:

PORT=5000
AI_API_KEY=your_api_key_here
JWT_SECRET=your_secret_key


(Adjust based on your actual backend implementation.)

🧪 Usage

Open the frontend in your browser

Authenticate (if enabled)

Start interacting with different AI models through the chatbot interface

🛠 Tech Stack

Frontend: JavaScript / Modern JS Framework (if applicable)

Backend: Node.js / Bun

Authentication: Token-based (JWT or similar)

AI Models: Multiple AI integrations (configurable)
